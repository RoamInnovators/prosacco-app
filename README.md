# ProSacco Mobile (Flutter)

Member-facing mobile app for ProSacco. API surface and UX notes live in `docs/member.md` (green/white theme, `/member` routes).

## Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable), with `flutter doctor` clean for your targets (Android / iOS / web).

## First-time setup (you run these)

From the repo root, use the folder name with a space in quotes:

```bash
cd "Mobile App"
flutter create . --project-name prosacco_mobile
flutter pub get
```

`flutter create .` adds `android/`, `ios/`, `web/`, etc., next to this `pubspec.yaml`.

## Run

```bash
cd "Mobile App"
flutter run
```

## Project layout (after `flutter create .`)

- `lib/` — Dart source (`main.dart` entry).
- Platform folders — generated by Flutter tooling.

Package name in `pubspec.yaml` is `prosacco_mobile` (valid for Dart); the on-disk folder remains `Mobile App` as requested.
